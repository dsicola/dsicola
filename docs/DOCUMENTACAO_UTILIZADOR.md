## 1. Administrador (ADMIN)

O Administrador tem acesso total à gestão da instituição.

### Principais funcionalidades

| Módulo | O que pode fazer |
|--------|------------------|
| **Dashboard** | Visão geral: estatísticas, alertas, acesso rápido |
| **Gestão Académica** | Cursos, turmas, disciplinas, turnos, classes |
| **Professores** | Cadastro, atribuição de disciplinas e turmas |
| **Alunos** | Cadastro, matrículas, histórico académico |
| **Matrículas** | Matricular alunos em turmas, transferências |
| **Planos de Ensino** | Criar, aprovar, distribuir aulas |
| **Lançamento de Aulas** | Registar aulas ministradas |
| **Presenças** | Lançar frequência dos alunos |
| **Avaliações e notas (disciplina)** | Configurar avaliações e lançar notas por disciplina / plano |
| **Finanças** | Mensalidades, pagamentos, bolsas, descontos |
| **Contabilidade** | Plano de contas, lançamentos, relatórios, exportação SAFT |
| **Configurações** | Dados da instituição, ano letivo, parâmetros |
| **Backups** | Agendar e gerar backups, restaurar |
| **Relatórios** | Pautas, boletins, estatísticas, exportações |

### Fluxos comuns

- **Início do ano letivo:** Configuração de Ensinos → Ano Letivo → Trimestres/Semestres → Cursos e Turmas → Matrículas
- **Lançamento por disciplina:** Avaliações e notas (disciplina) → contexto (curso/classe, disciplina, professor) → separador Avaliações / Lançamento de Notas.
- **Visão por turma:** Gestão Académica → Notas (ou **Notas e pautas (turma)** no menu); seleccionar turma e plano quando houver várias disciplinas.
- **Gerar mensalidades:** Finanças → Gerar Mensalidades (mensal)

## 2. Secretaria

A Secretaria apoia a administração nas operações do dia a dia.

### Principais funcionalidades

| Módulo | O que pode fazer |
|--------|------------------|
| **Dashboard** | Visão geral e tarefas pendentes |
| **Alunos** | Consultar, cadastrar, editar dados |
| **Matrículas** | Matricular alunos, imprimir comprovantes |
| **Presenças** | Consultar e lançar frequência |
| **Notas** | Consultar pautas e boletins |
| **Documentos** | Emitir declarações, certificados |
| **Finanças** | Consultar mensalidades, registar pagamentos |
| **Comunicados** | Enviar comunicados às turmas |

### Diferença em relação ao Admin

- Não acede a Configurações da instituição
- Não cria/edita ano letivo ou estrutura académica
- Não gere backups

## 3. Professor

O Professor gere as suas turmas e disciplinas.

### Principais funcionalidades

| Módulo | O que pode fazer |
|--------|------------------|
| **Dashboard** | Turmas, aulas do dia, avisos |
| **Minhas Turmas** | Ver turmas e disciplinas atribuídas |
| **Plano de Ensino** | Consultar planos aprovados (apenas leitura) |
| **Lançar Aulas** | Registar aulas ministradas |
| **Frequência** | Marcar presenças e faltas |
| **Notas (plano + turma)** (Painel professor) | Lançamento integrado turma/disciplina do plano |
| **Biblioteca** | Consultar acervo e solicitar empréstimos |

### O que o Professor não pode fazer

- Criar ou editar planos de ensino
- Configurar ano letivo ou estrutura académica
- Aceder a dados de outras turmas/disciplinas
- Gerir mensalidades ou configurações da instituição

## 6. Financeiro

O perfil Financeiro gere a parte económica da instituição.

### Principais funcionalidades

| Módulo | O que pode fazer |
|--------|------------------|
| **Finanças** | Mensalidades, pagamentos, bolsas, descontos |
| **Contabilidade** | Plano de contas, lançamentos, relatórios (Balancete, Balanço, DRE), fecho de exercício, exportação Excel/SAFT |

## Suporte técnico

- **Problemas de acesso:** Contacte o administrador da instituição.
- **Esqueci a senha:** Use "Recuperar senha" na página de login.
- **2FA:** Se ativado, use a aplicação autenticadora (Google Authenticator, etc.) para o código.
